Explain Like I Am Nerd

Alice installs Tessercube.

Upon initialization, an ECC-256 keypair is generated automatically.

Alice publishes her pubkey in a post or in her bio.

Bob, a friend of Alice, follows the same procedure.

When Alice posts, the content is encrypted with a random key via AES-256.

Alice's Tessercube encrypts the AES key with Bob's pubkey via ECDH.

Alice's Tessercube sends the encrypted AES key to Bob over Peer to peer data transfer.

Bob's Tessercube decrypts the message and gets the AES key.

Bob's Tessercube decrypts the post with the AES key for Bob. Task done.

For specific details, see Documentation on GitHub.
